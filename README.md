## Technologies Used
- Python 3.x
- requests
- BeautifulSoup

## How to Run
1. Install Python 3.x
2. Install dependencies:
   pip install -r requirements.txt
3. Run the script:
   python scraper.py

## Output
The script generates an `output.json` file containing the scraped records.

## Notes
